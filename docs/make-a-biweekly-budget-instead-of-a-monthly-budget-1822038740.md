# 做一个两周的预算，而不是一个月的预算

> 原文：<https://lifehacker.com/make-a-biweekly-budget-instead-of-a-monthly-budget-1822038740>

努力坚持每月预算(或者甚至一开始就制定预算)？考虑创建一个双周预算。如果你每两周(或每月两次)领一次工资，你可能会更容易用这种方式管理你的钱，因为它考虑到了没有一周的支出是相同的:有些周你可能会格外节俭和负责任，而其他周你可能会有意想不到的支出或决定犒劳自己。

如果你的预算是两周一次，它可以直接与薪水相吻合，让你在精神上“重新开始”，比一个月的预算更快。它也承认我们应该更灵活地使用我们的钱。

### 如何制定双周预算

*   在一张工资单上写下典型的实得工资金额。
*   仔细查看你的银行对账单，找出你每月的所有固定支出——例如，房租/抵押贷款、汽车贷款、有线电视/互联网、食品杂货和娱乐活动— ,然后将每个数字除以 2。

*   从你的薪水中减去总数，然后把它存入一个储蓄账户。(你也可以两周支付一次账单，比如你的信用卡，以提前还款。)
*   减去任何“目标”资金，比如你的储蓄账户或度假基金(就我个人而言，我在我的计划中写了一个目标金额，所以我记得把钱存进去)，然后把它存入一个不同的储蓄账户。
*   剩下的就是你接下来两周要花的钱了。

按照这个模板，你将一年中的 24 份工资入账。如果你两周领一次工资，那么实际上你每年会得到 26 张支票，而且你可以把这些“奖金”存起来用于偿还债务，或者用于抵押贷款，或者满足其他财务目标。

它还允许你提前为更大的账单和目标留出资金，而不是用一份薪水支票，比如说，你的“租金”支票，并试图让另一份贯穿整个月。

还有其他方法可以做到这一点——比如，你需要一份预算 建议从每月的第一份薪水中扣除所有的固定支出(比如房租和车贷)，这样你就可以更灵活地使用第二份薪水。但这一切都归结于以一种对你有意义的方式管理你的钱。
