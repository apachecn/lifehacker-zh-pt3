# Waze 仍然提供最快的驾驶路线

> 原文:[https://life hacker . com/waze-still-provide-the-fast-driving-directions-1823286419](https://lifehacker.com/waze-still-provides-the-fastest-driving-directions-1823286419)

如果你想快速到达目的地，那么你最好使用谷歌地图或 Waze，而不是苹果地图——如果你真的想要“最快”的选项，就使用 Waze。

Watch

Adobe 的一位经理进行的非正式测试发现，苹果公司的预计行程时间通常比实际行程时间长 8%。这位名叫阿图尔·格拉博斯基的工程师表示，他花了 2017 年的“大部分时间”在苹果地图、谷歌地图和 Waze 上收集数据。

虽然预计的旅行时间相差 8%，但使用苹果地图导航通常会比软件预测的到达时间快 1%。

格拉博斯基说“苹果公司夸大了它的估计，这样用户平均会在预测的时间到达或者稍微早一点。”所以基本上它是故意告诉你一次旅行会比实际花费的时间长。

它给你一个保守的估计，所以当你比预期早一点到达时，你会有一个很好的体验，但方向本身往往不如你从 Waze 或谷歌地图上获得的那样快。

也就是说，如果速度是你想要的，那么谷歌地图是一个更好的选择。谷歌地图的导航速度比苹果地图快 5%。如果你换成谷歌旗下的 Waze，事情会变得更快。使用 Waze 导航比苹果地图快 6%。
