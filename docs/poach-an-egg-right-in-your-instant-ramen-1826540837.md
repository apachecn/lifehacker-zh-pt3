# 在你的方便面里煮一个鸡蛋

> 原文：<https://lifehacker.com/poach-an-egg-right-in-your-instant-ramen-1826540837>

速食拉面可能是最完美的廉价食品。它本身很好，但是总有创造性的方法把它变成更好的东西 。我最喜欢做的事情是在煮拉面的时候往锅里打一个鸡蛋。

Watch

在美味的韩国辣软豆腐炖菜，或 soondubu jjigae (순두부 찌개)成为我外出就餐的常规食物后，我开始这样做。当他们给你端上热气腾腾的炖肉时，总是会附带一个生鸡蛋，让你敲开后放进去，这给这道菜增添了一层额外的风味和丰富。它是如此美味，这让我想到了其他可以受益于相同概念的食物。

谈到拉面，我有时会在碗里加入自制的酱油调味的软蛋 或 shoyu tamago，但我想要 soondubu jjigae 中鸡蛋的细腻奶油。所以我想为什么不在家里对我的拉面做同样的事情呢？

## 如何在方便面里煮一个鸡蛋

1.  我放上一小壶水烧开。

2.  我把拉面调料加入水中。我可能还会加一点比肉汤更好的东西。

3.  我加入面条，然后慢炖。

4.  一旦面条稍微变软，我就在拉面纠结的砖块中间开一个小口，然后在里面打一个鸡蛋。这就像是鸡蛋的摇篮，所以它不会在你的肉汤里立刻破碎。

5.  我会添加任何我喜欢的额外食物，比如蔬菜或肉片。
6.  我重新加热一会儿，让鸡蛋煮熟，然后再炖一两分钟，让面条变软。

就是这样！我把所有东西都倒进碗里，然后开始吃。用你的剁碎的棍子或叉子快速戳一下鸡蛋，就会把奶油状的、松软的蛋黄送到肉汤里。试试看。
